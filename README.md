# Overall Commercial Performance - Fit to Screen Version

This version keeps the full table visible on screen as much as possible by:

- compacting the table layout
- automatically scaling the table to the available screen area
- removing page scroll in table view
- keeping arrow-key movement and Firestore save/load

How to use:

1. Open `index.html`
2. Select region and year
3. Press **Next**
4. Enter data
5. Press **Save to Cloud**

Notes:

- For best results, use the browser window maximized.
- This version is designed for desktop/laptop use.
- The table automatically resizes when the window size changes.
